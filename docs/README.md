# norex Documentation

## About norex

norex is an experimental operating system developed for learning and exploration of low-level systems programming, kernel development, and OS architecture.

## Documentation Structure

This documentation covers:

- **Getting Started** - Setting up your development environment
- **Architecture** - Overview of the norex OS architecture
- **Kernel** - Core kernel components and internals
- **Syscalls** - System call interface and ABI
- **Usermode** - Userspace development and process management
- **GUI System** - Graphics and windowing system
- **Memory Management** - Physical and virtual memory management
- **Building & Testing** - How to build and test norex
- **Contributing** - Guidelines for contributing to the project
