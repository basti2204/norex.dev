What is norex?

norex is a custom operating system developed from the ground up. It features a protected kernel, a real usermode, syscalls, multitasking and a custom graphical interface.

The project focuses on understanding operating systems at the lowest level — not abstraction, but control.

Core Features

  • Protected kernel (Ring 0)
  • Usermode processes (Ring 3)
  • Custom syscall interface
  • Multitasking & scheduler
  • Virtual memory & paging
  • Framebuffer-based GUI
