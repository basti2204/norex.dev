Getting Started

Ready to explore norex? Whether you want to run it or build it from source, here's how to begin.

System Requirements

  • CPU: x86-64 compatible processor
  • RAM: Minimum 512MB
  • Boot: UEFI or BIOS support
  • Storage: 100MB+ available space

Quick Start

  • Download the latest ISO image
  • Create a bootable USB or VM
  • Boot norex and explore
  • Check the docs for tutorials

Build From Source

  • Clone the repository (when public)
  • Install build dependencies
  • Run the build script
  • Test in QEMU or real hardware
