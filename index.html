Why norex?

norex is not designed to compete with Linux or Windows. It exists to explore how modern operating systems actually work — from scheduling and paging down to syscalls and hardware control.

Core Features

Protected Kernel

Strict Ring 0 / Ring 3 separation with enforced memory protection.

Usermode

Real userspace processes communicating via syscalls.

Multitasking

Preemptive scheduler with isolated tasks.

Virtual Memory

Paging, address spaces and memory safety.

Custom GUI

Framebuffer-based window system without external toolkits.
